# SR Garments - Full-Stack E-Commerce Platform

A modern, high-performance clothing store application. This project demonstrates a clean separation of concerns between a dynamic React frontend and a scalable Node/Express backend.

## 🛠 Technical Architecture

### Frontend (Client-Side)
* **React.js (Functional Components & Hooks):** Leveraged for building a fast, modular user interface.
* **Redux Toolkit (RTK):** Implemented for sophisticated state management (handling cart logic, user sessions, and product caching).
* **Bootstrap 5:** Utilized for a "mobile-first" responsive design and professional UI components.
* **Axios:** Used for seamless asynchronous API communication with the backend.

### Backend (Server-Side)
* **Express.js:** A streamlined web framework for Node.js used to build a secure RESTful API.
* **Middleware Integration:** Custom error handling and data validation routines.
* **REST API Design:** Structured endpoints for product management, user authentication, and order processing.
* **Environment Configuration:** Secure management of API keys and database strings using `dotenv`.

---

## 🚀 Key Features Under Development

* **State-Driven Cart:** Real-time updates across the app using Redux slices.
* **Dynamic Product Grid:** Fetching and filtering garment data from the Express backend.
* **Secure Auth:** Implementation of JWT-based authentication for user accounts.
* **Responsive Layout:** Fully optimized for all screen sizes using Bootstrap's grid system.
